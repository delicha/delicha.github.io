個人開発実績

Rails+Nginx+Docker+AWS 印象評価アプリの開発

https://scouter.fun/

こちらのアプリのQiita記事
(LGTM100以上/24000views以上)

▶︎ 使用技術

　HTML | CSS | SCSS | JavaScript | Ruby | Ruby on Rails | SQL | Nginx | AWS（VPC／EC2／Route53／IGW／ALB／ACM／RDS／S3／CloudFront）| Git | GitHub

▶︎ ポイント

  • ・リアルなサービスに近いデザイン（レスポンシブ対応）
  • ・マッチングアプリのマッチングするまでを補うようなアプリ。自分の印象の評価が男女別、世代別、また相手の性格や趣味など多角的に分析できる
  • ・パブリックサブネットを2つ用意しロードバランサーを据え可用性を高めたインフラ構築
  • ・開発環境、本番環境でシステム構成を分けて、それぞれDockerにて管理
  • ・AWSのCloud Front、S3、Route53、ACM、RDS、IAMを利用
  • ・Deviseを使ったログイン認証、ユーザーと管理者を分け、管理者画面を設置
  • ・DBの画像をカルーセルで表示、また個別に削除する機能
  • ・個人開発だがGithubでブランチを切り、プルリクしながら開発
  • ・独自ドメイン取得
